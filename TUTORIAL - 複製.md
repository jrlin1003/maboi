## 📖 網頁三劍客簡介

### 1. HTML（超文本標記語言）
- **作用**：網頁的「骨架」，定義內容和結構
- **檔案**：`index.html`
- **比喻**：就像房子的鋼筋水泥框架

**常用標籤：**
```html
<h1>標題</h1>          <!-- 大標題 -->
<p>段落</p>            <!-- 文字段落 -->
<img src="圖片網址">    <!-- 圖片 -->
<a href="連結">文字</a> <!-- 超連結 -->
<button>按鈕</button>   <!-- 按鈕 -->
<div>容器</div>        <!-- 容器，用來組織內容 -->
```

### 2. CSS（階層樣式表）
- **作用**：網頁的「外觀」，負責顏色、大小、排版
- **檔案**：`style.css`
- **比喻**：就像房子的油漆、裝潢和家具

**常用屬性：**
```css
color: blue;              /* 文字顏色 */
background-color: red;    /* 背景顏色 */
font-size: 20px;          /* 文字大小 */
padding: 10px;            /* 內距（內部空間） */
margin: 20px;             /* 外距（外部空間） */
border-radius: 10px;      /* 圓角 */
```

### 3. JavaScript（程式語言）
- **作用**：網頁的「互動功能」，讓網頁能回應使用者
- **檔案**：`script.js`
- **比喻**：就像房子的電燈開關、自動門

**常用功能：**
```javascript
// 選取元素
const button = document.getElementById('myButton');

// 監聽點擊事件
button.addEventListener('click', function() {
    alert('你點擊了按鈕！');
});
```

## 🎨 開始客製化你的網頁

### 任務 1️⃣：修改個人資訊

1. 開啟 `index.html`
2. 找到「王小明」的地方（按 `Ctrl+F` 搜尋）
3. 改成你自己的名字
4. 修改關於我的介紹內容
5. **存檔**（`Ctrl+S`）並看預覽畫面的變化！

### 任務 2️⃣：更換配色

打開 `style.css`，找到最上面的 CSS 變數：

```css
:root {
    --primary-color: #6366f1;  /* 主色調 */
}
```

試試改成其他顏色：
- 紅色：`#ef4444`
- 綠色：`#10b981`
- 橙色：`#f59e0b`
- 粉色：`#ec4899`

**💡 提示**：修改後立刻存檔，網頁會自動更新！

### 任務 3️⃣：試試深色模式

點擊右上角的 🌙 按鈕，看看網頁切換到深色模式！

這是怎麼做到的？打開 `script.js` 找到這段程式碼：

```javascript
themeToggle.addEventListener('click', function() {
    document.body.classList.toggle('dark-mode');
    // ... 更多程式碼
});
```

## 🔧 常用操作

### 檔案管理

- **開啟檔案**：點擊左側檔案清單
- **搜尋內容**：`Ctrl+F`（在單一檔案中）或 `Ctrl+Shift+F`（在整個專案）

- **復原上一步**：`Ctrl+Z`
- **重做**：`Ctrl+Y`
- **查看變更記錄**：點擊左側的「Source Control」圖示

## 💡 重要概念解釋

### 什麼是「選取器」（Selector）？

在 CSS 中，選取器用來指定要套用樣式的元素：

```css
/* 標籤選取器：套用到所有 <p> */
p {
    color: blue;
}

/* 類別選取器：套用到 class="highlight" */
.highlight {
    background-color: yellow;
}

/* ID 選取器：套用到 id="header" */
#header {
    font-size: 24px;
}
```

### 什麼是「盒模型」（Box Model）？

每個 HTML 元素都是一個盒子：

```
┌─────────────────────────────────┐
│         margin (外距)            │
│  ┌───────────────────────────┐  │
│  │     border (邊框)         │  │
│  │  ┌─────────────────────┐  │  │
│  │  │  padding (內距)     │  │  │
│  │  │  ┌───────────────┐  │  │  │
│  │  │  │   content     │  │  │  │
│  │  │  │   (內容)      │  │  │  │
│  │  │  └───────────────┘  │  │  │
│  │  └─────────────────────┘  │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

### 什麼是「事件監聽」（Event Listener）？

JavaScript 用來等待使用者動作的機制：

```javascript
// 等待點擊事件
button.addEventListener('click', function() {
    // 點擊時執行這裡的程式碼
});

// 常見事件類型：
// 'click' - 點擊
// 'mouseover' - 滑鼠移入
// 'submit' - 表單送出
// 'scroll' - 捲動
```

## ❓ 常見問題

### Q1：我改了程式碼，但網頁沒有變化？
**A**：
1. 確認你有存檔（`Ctrl+S`）
2. 試試重新整理預覽畫面
3. 檢查是否有拼字錯誤

### Q2：我的程式碼跑出錯誤訊息！
**A**：
1. 看看錯誤訊息，通常會指出是哪一行
2. 檢查是否缺少分號 `;` 或括號 `{}`
3. 使用 `Ctrl+Z` 復原到上一個正常的狀態
4. 請 AI 協助檢查錯誤

### Q3：如何選擇好看的配色？
**A**：
1. 參考這些網站：
   - [Coolors](https://coolors.co/) - 配色產生器
   - [Color Hunt](https://colorhunt.co/) - 配色靈感
2. 或直接問 AI：「請給我 5 組適合個人網頁的配色方案」

### Q4：手機版看起來怪怪的？
**A**：
1. 檢查 CSS 中的 `@media` 響應式設定
2. 使用瀏覽器開發者工具的手機模擬器測試
3. 請 AI 協助：「請優化手機版的顯示」

### Q5：我想要的功能太複雜，AI 做不出來？
**A**：
1. 把大功能拆成小步驟
2. 一次只要求一個小功能
3. 參考 `AI-PROMPTS.md` 學習如何分解需求
