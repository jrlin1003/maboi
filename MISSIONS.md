## 🎯 任務 1：認識自己的網頁 ⭐

### 目標
熟悉專案中的三個主要檔案，並做簡單的修改。

### 步驟
1. 開啟 `index.html`，找到你的名字並改成真實姓名（或筆名）
2. 修改「關於我」區塊的自我介紹文字

4. 修改聯絡資訊（email、Instagram 帳號等）

### 檢查點

## 🎨 任務 2：打造專屬配色 ⭐

### 目標
學習使用 CSS 變數，設計一套專屬的配色方案。

### 步驟
1. 開啟 `style.css`，找到最上方的 `:root` 區塊
2. 修改 `--primary-color`（主色調）
3. 試試不同顏色，觀察整個網頁的變化
4. 調整其他顏色變數，打造完整配色方案

### 顏色建議
- **活力橙**：`#f97316`
- **清新綠**：`#10b981`
- **典雅紫**：`#a855f7`
- **科技藍**：`#3b82f6`
- **夢幻粉**：`#ec4899`

### 檢查點
- ✅ 成功修改主題顏色
- ✅ 理解 CSS 變數的概念
- ✅ 嘗試至少 3 種不同配色

### AI 提示詞範例
```
請推薦 5 組適合個人網頁的配色方案，
並告訴我如何在 style.css 中修改
```

## 🚀 任務 6：返回頂部按鈕 ⭐

### 目標
加上一個「返回頂部」按鈕，讓使用者能快速回到頁面最上方。

### 需求
1. 按鈕固定在右下角
2. 只在使用者往下捲動超過 300px 時才出現
3. 點擊後平滑捲動回頂部

### 設計提示
- 使用向上箭頭圖示：`↑` 或 `⬆️`
- 按鈕要有陰影和懸停效果
- 考慮深色模式下的按鈕顏色

### 檢查點
- ✅ 按鈕出現/消失功能正常
- ✅ 點擊後能平滑回到頂部
- ✅ 設計美觀且不遮擋內容

### AI 提示詞範例
```
請在網頁右下角加上一個「返回頂部」按鈕，
需求：
1. 固定在右下角
2. 當捲動超過 300px 時淡入顯示
3. 點擊後平滑捲動回頁面頂部
```

## 🌟 任務 9：建立專屬主題切換 ⭐⭐⭐

### 目標
擴充現有的深色模式，新增多個主題選項供使用者選擇。

### 需求
1. 至少 3 個主題：淺色、深色、護眼綠
2. 主題選單可以讓使用者選擇
3. 記住使用者的選擇（使用 localStorage）
4. 切換主題時有平滑過渡效果

### 主題建議
- **經典淺色**：預設的白色主題
- **午夜深色**：深藍黑色系
- **護眼綠**：淡綠色背景，降低藍光
- **櫻花粉**：粉色系主題
- **極光藍**：藍紫漸層主題

### 檢查點
- ✅ 能切換至少 3 個主題
- ✅ 重新開啟網頁時保留主題選擇
- ✅ 主題切換動畫流暢

### AI 提示詞範例
```
請擴充現有的深色模式功能，改為多主題切換系統：

1. 新增主題選單（取代原本的月亮按鈕）
2. 提供 4 個主題選項：
   - 經典淺色
   - 午夜深色
   - 護眼綠
   - 櫻花粉

3. 每個主題定義完整的配色方案（背景、文字、主色等）
4. 點擊主題選單時顯示下拉選項
5. 選擇主題後儲存到 localStorage
6. 加上主題切換的過渡動畫

請先在 CSS 定義各主題的變數，再用 JavaScript 實作切換邏輯。
```
