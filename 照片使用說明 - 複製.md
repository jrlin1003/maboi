## 🎯 如何添加照片到網頁

### 方法一：使用本地檔案（推薦）

**步驟：**
1. 將照片複製到 `images` 資料夾
2. 確認檔案名稱（例如：`my-photo.jpg`）
3. 在 HTML 中使用相對路徑

**範例：**
```html
<!-- 照片在 images 資料夾內 -->
<img src="images/my-photo.jpg" alt="我的照片">

<!-- 照片與 index.html 在同一層 -->
<img src="photo.jpg" alt="照片">
```

### 方法二：使用網路圖片

```html
<!-- 直接使用圖片網址 -->
<img src="https://example.com/photo.jpg" alt="照片">

<!-- 目前使用的 Unsplash（免費高品質圖片）-->
<img src="https://images.unsplash.com/photo-xxx?w=400&h=300&fit=crop" alt="照片">
```

## 🖼️ 完整的 `<img>` 標籤屬性

```html
<img 
    src="images/photo.jpg"        <!-- 必填：圖片路徑 -->
    alt="照片描述"                 <!-- 必填：替代文字（重要！）-->
    width="400"                   <!-- 選填：寬度 -->
    height="300"                  <!-- 選填：高度 -->
    class="gallery-image"         <!-- 選填：CSS 樣式類別 -->
    loading="lazy"                <!-- 選填：延遲載入（提升效能）-->
>
```

### 重要屬性說明

- **`src`**：圖片的位置（檔案路徑或網址）
- **`alt`**：圖片的文字描述
  - 當圖片載入失敗時會顯示
  - 螢幕閱讀器會讀出來（無障礙重要）
  - SEO 優化必備

## ❓ 常見問題

### Q1: 照片不顯示怎麼辦？

**檢查清單：**
- ✅ 檔案路徑是否正確？
- ✅ 檔案名稱大小寫是否一致？
- ✅ 照片檔案是否確實存在？
- ✅ 檔案格式是否支援？（.jpg, .png, .gif, .webp）

**正確的路徑寫法：**
```html
<!-- ✅ 正確 -->
<img src="images/photo.jpg">

<!-- ❌ 錯誤 -->
<img src="image/photo.jpg">     <!-- 資料夾名稱錯誤 -->
<img src="images\photo.jpg">    <!-- 使用反斜線（Windows 路徑）-->
<img src="images/Photo.jpg">    <!-- 大小寫不一致 -->
```

### Q2: 如何壓縮照片？

**線上工具（免費）：**
- TinyPNG - https://tinypng.com/
- Squoosh - https://squoosh.app/
- Compressor.io - https://compressor.io/

**Windows 內建：**
1. 用「小畫家」開啟照片
2. 調整影像大小
3. 另存新檔

### Q3: 可以放影片嗎？

可以！使用 `<video>` 標籤：

```html
<video width="400" controls>
    <source src="images/my-video.mp4" type="video/mp4">
    您的瀏覽器不支援影片播放。
</video>
```

## 🚀 快速範例：新增一張照片

**完整步驟：**

1. **準備照片**
   ```
   將 my-new-photo.jpg 複製到 images 資料夾
   ```

2. **在 HTML 中添加**
   ```html
   <div class="gallery-item">
       <img src="images/my-new-photo.jpg" 
            alt="我的新照片" 
            class="gallery-image">
       <div class="gallery-overlay">
           <p>照片標題</p>
       </div>
   </div>
   ```

3. **儲存並重新整理網頁**
   ```
   按 Ctrl + S 儲存
   在瀏覽器按 F5 重新整理
   ```

完成！🎉

## 💡 最佳實踐建議

1. **檔案命名規則**
   - ✅ 使用英文和數字：`profile-photo.jpg`
   - ✅ 使用小寫：`sunset.jpg`
   - ✅ 用減號分隔：`city-night.jpg`
   - ❌ 避免中文：`我的照片.jpg`
   - ❌ 避免空格：`my photo.jpg`

2. **效能優化**
   - 壓縮圖片檔案大小
   - 使用 `loading="lazy"` 延遲載入
   - 選擇合適的圖片格式（WebP 更小）

3. **無障礙設計**
   - 一定要寫 `alt` 屬性
   - 描述要清楚具體
   - 裝飾性圖片可用 `alt=""`

4. **響應式設計**
   - CSS 中設定 `max-width: 100%`
   - 使用 `<picture>` 標籤支援多尺寸
   - 考慮不同裝置的顯示效果
