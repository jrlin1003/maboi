## 📚 第一單元：認識網頁三劍客

### 什麼是 HTML？

**定義**：HyperText Markup Language（超文本標記語言）

**比喻**：HTML 就像是房屋的「骨架」和「結構」
- 決定網頁有哪些元素
- 定義內容的類型（標題、段落、圖片等）
- 不負責外觀美化

**實際範例**：
```html
<!DOCTYPE html>
<html lang="zh-TW">
<head>
    <meta charset="UTF-8">
    <title>我的第一個網頁</title>
</head>
<body>
    <h1>哈囉！我是王小明</h1>
    <p>歡迎來到我的個人網頁</p>
</body>
</html>
```

**學習重點**：
- `<!DOCTYPE html>` 告訴瀏覽器這是 HTML5 文件
- `<html>` 是根元素，包含整個網頁
- `<head>` 放置網頁的設定資訊
- `<body>` 放置實際顯示的內容

---

### 什麼是 CSS？

**定義**：Cascading Style Sheets（層疊樣式表）

**比喻**：CSS 就像是房屋的「裝潢」和「美化」
- 控制顏色、字體、大小
- 設定排版和間距
- 製作動畫效果

**實際範例**：
```css
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
}

h1 {
    color: #6366f1;
    font-size: 2.5rem;
    text-align: center;
}

p {
    color: #666;
    line-height: 1.6;
}
```

**學習重點**：
- 選擇器（selector）：選擇要套用樣式的元素
- 屬性（property）：要改變的樣式項目
- 值（value）：樣式的具體設定

---

### 什麼是 JavaScript？

**定義**：網頁的程式語言

**比喻**：JavaScript 就像是房屋的「智慧家電」
- 讓網頁能夠互動
- 回應使用者的操作
- 動態改變內容

**實際範例**：
```javascript
// 當按下按鈕時，顯示訊息
document.getElementById('myButton').addEventListener('click', function() {
    alert('你好！歡迎來到我的網頁');
});

// 深色模式切換
function toggleDarkMode() {
    document.body.classList.toggle('dark-mode');
}
```

**學習重點**：
- 可以監聽事件（點擊、滾動、輸入等）
- 可以改變 HTML 內容和 CSS 樣式
- 可以進行計算和邏輯判斷

---

### 三者如何協同運作？

**建築比喻總整理**：
- **HTML** = 房屋結構（牆壁、窗戶、門）
- **CSS** = 室內裝潢（油漆、家具、擺設）
- **JavaScript** = 智慧家電（自動門、智慧燈光）

**實際運作流程**：
1. 瀏覽器讀取 HTML 檔案
2. 遇到 `<link rel="stylesheet" href="style.css">` 就載入 CSS
3. 遇到 `<script src="script.js">` 就執行 JavaScript
4. 三者結合呈現完整的網頁

**檔案結構範例**：
```
我的網站/
├── index.html      ← 主要結構
├── style.css       ← 樣式設定
└── script.js       ← 互動功能
```

## 📝 第二單元：建立第一個 HTML 網頁

### HTML 的基本結構

**必備元素**：
```html
<!DOCTYPE html>           <!-- 宣告文件類型 -->
<html lang="zh-TW">      <!-- 網頁的根元素，lang 設定語言 -->
<head>                   <!-- 網頁的頭部資訊 -->
    <meta charset="UTF-8">                    <!-- 字元編碼 -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">  <!-- 響應式設計必備 -->
    <title>網頁標題</title>                    <!-- 瀏覽器分頁顯示的標題 -->
    <link rel="stylesheet" href="style.css">  <!-- 連結 CSS 檔案 -->
</head>
<body>                   <!-- 網頁的主體內容 -->
    <!-- 這裡放置所有可見的內容 -->
    
    <script src="script.js"></script>         <!-- 連結 JavaScript 檔案 -->
</body>
</html>
```

---

### 常用 HTML 標籤介紹

#### 1. 標題標籤（h1 到 h6）

```html
<h1>最大的標題（通常一頁只用一個）</h1>
<h2>第二層標題</h2>
<h3>第三層標題</h3>
<h4>第四層標題</h4>
<h5>第五層標題</h5>
<h6>最小的標題</h6>
```

**使用時機**：
- h1：網頁主標題（如：王小明的個人網頁）
- h2：區塊標題（如：關於我、作品集）
- h3：小標題（如：我的興趣、聯絡方式）

---

#### 2. 段落與文字標籤

```html
<p>這是一個段落。段落會自動換行。</p>

<strong>粗體文字（強調）</strong>
<em>斜體文字（強調）</em>
<br>  <!-- 換行 -->

<span>行內文字容器</span>
<div>區塊容器</div>
```

---

#### 3. 列表標籤

```html
<!-- 無序列表（項目符號）-->
<ul>
    <li>項目一</li>
    <li>項目二</li>
    <li>項目三</li>
</ul>

<!-- 有序列表（數字編號）-->
<ol>
    <li>第一步</li>
    <li>第二步</li>
    <li>第三步</li>
</ol>
```

---

#### 4. 連結與圖片

```html
<!-- 超連結 -->
<a href="https://www.google.com">前往 Google</a>
<a href="#about">跳到關於我區塊</a>
<a href="mailto:email@example.com">寄信給我</a>

<!-- 圖片 -->
<img src="images/photo.jpg" alt="照片描述">
```

**圖片標籤重要屬性**：
- `src`：圖片路徑（必填）
- `alt`：替代文字（必填，無障礙重要）
- `width`、`height`：尺寸（選填）

---

#### 5. 語意化標籤（HTML5）

```html
<nav>       <!-- 導覽列 -->
    <a href="#home">首頁</a>
    <a href="#about">關於</a>
</nav>

<header>    <!-- 頁首 -->
    <h1>網站標題</h1>
</header>

<main>      <!-- 主要內容 -->
    <section id="about">   <!-- 區塊 -->
        <h2>關於我</h2>
        <p>介紹文字</p>
    </section>
    
    <article>  <!-- 文章 -->
        <h3>文章標題</h3>
        <p>文章內容</p>
    </article>
</main>

<footer>    <!-- 頁尾 -->
    <p>版權資訊</p>
</footer>
```

**為什麼要用語意化標籤？**
1. **SEO 優化**：搜尋引擎更容易理解網頁結構
2. **無障礙**：螢幕閱讀器能正確朗讀
3. **可維護性**：程式碼更容易閱讀和維護

---

#### 6. 表單標籤

```html
<form id="contactForm">
    <!-- 文字輸入 -->
    <input type="text" placeholder="你的名字" required>
    
    <!-- Email 輸入 -->
    <input type="email" placeholder="你的 Email" required>
    
    <!-- 多行文字 -->
    <textarea placeholder="留言內容" rows="5"></textarea>
    
    <!-- 送出按鈕 -->
    <button type="submit">送出</button>
</form>
```

## 🎨 第三單元：CSS 美化網頁

### CSS 基本語法

**結構**：
```css
選擇器 {
    屬性: 值;
    屬性: 值;
}
```

**實際範例**：
```css
h1 {
    color: blue;           /* 文字顏色 */
    font-size: 2rem;       /* 字體大小 */
    text-align: center;    /* 文字對齊 */
}
```

---

### 三種 CSS 引入方式

#### 1. 外部樣式表（推薦）

```html
<!-- HTML 檔案中 -->
<link rel="stylesheet" href="style.css">
```

```css
/* style.css 檔案中 */
body {
    background-color: #f0f0f0;
}
```

**優點**：
- 多個 HTML 可共用同一個 CSS
- 易於維護和管理
- 符合關注點分離原則

---

#### 2. 內部樣式表

```html
<head>
    <style>
        body {
            background-color: #f0f0f0;
        }
    </style>
</head>
```

**適用時機**：單一頁面的特殊樣式

---

#### 3. 行內樣式

```html
<p style="color: red; font-size: 18px;">這是紅色文字</p>
```

**缺點**：
- 難以維護
- 無法重複使用
- 不建議使用

---

### CSS 選擇器

#### 1. 元素選擇器

```css
h1 {
    color: blue;
}

p {
    font-size: 16px;
}
```

**作用**：選擇所有該類型的元素

---

#### 2. 類別選擇器（Class）

```html
<div class="card">卡片內容</div>
<div class="card highlight">特殊卡片</div>
```

```css
.card {
    background-color: white;
    padding: 20px;
    border-radius: 10px;
}

.highlight {
    border: 2px solid gold;
}
```

**特點**：
- 可重複使用
- 一個元素可以有多個 class
- 用 `.` 開頭

---

#### 3. ID 選擇器

```html
<div id="header">這是頁首</div>
```

```css
#header {
    background-color: navy;
    color: white;
}
```

**特點**：
- 一個頁面中 ID 必須唯一
- 用 `#` 開頭
- 優先權高於 class

---

#### 4. 組合選擇器

```css
/* 後代選擇器：選擇 nav 內的所有 a */
nav a {
    color: blue;
}

/* 子元素選擇器：只選擇直接子元素 */
.nav-menu > li {
    display: inline-block;
}

/* 群組選擇器：同時套用多個元素 */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

/* 偽類選擇器：滑鼠懸停效果 */
a:hover {
    color: red;
}
```

---

### CSS 變數（Custom Properties）

**為什麼需要 CSS 變數？**
- 統一管理顏色配置
- 輕鬆實現深色模式
- 提高可維護性

**定義與使用**：
```css
:root {
    /* 定義變數 */
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --text-color: #1f2937;
    --bg-color: #ffffff;
}

/* 使用變數 */
body {
    color: var(--text-color);
    background-color: var(--bg-color);
}

h1 {
    color: var(--primary-color);
}

.btn {
    background-color: var(--primary-color);
}

.btn:hover {
    background-color: var(--secondary-color);
}
```

**深色模式實作**：
```css
/* 淺色模式（預設）*/
:root {
    --text-color: #1f2937;
    --bg-color: #ffffff;
}

/* 深色模式 */
body.dark-mode {
    --text-color: #f9fafb;
    --bg-color: #111827;
}
```

### 常用 CSS 屬性

#### 文字相關

```css
.text {
    /* 字體 */
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;          /* normal, bold, 100-900 */
    
    /* 顏色與對齊 */
    color: #333;
    text-align: center;         /* left, center, right, justify */
    line-height: 1.6;           /* 行高 */
    
    /* 裝飾 */
    text-decoration: underline; /* none, underline, line-through */
}
```

---

#### 背景相關

```css
.box {
    /* 單色背景 */
    background-color: #f0f0f0;
    
    /* 圖片背景 */
    background-image: url('images/bg.jpg');
    background-size: cover;      /* 覆蓋整個區域 */
    background-position: center; /* 置中 */
    background-repeat: no-repeat;
    
    /* 漸層背景 */
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}
```

---

#### 盒模型（Box Model）

```css
.box {
    /* 內容寬高 */
    width: 300px;
    height: 200px;
    
    /* 內距（內容與邊框的距離）*/
    padding: 20px;               /* 四邊都是 20px */
    padding: 10px 20px;          /* 上下 10px，左右 20px */
    padding: 10px 20px 30px 40px; /* 上 右 下 左（順時針）*/
    
    /* 邊框 */
    border: 2px solid #333;      /* 寬度 樣式 顏色 */
    border-radius: 10px;         /* 圓角 */
    
    /* 外距（與其他元素的距離）*/
    margin: 20px;
    margin: 0 auto;              /* 水平置中 */
}
```

**盒模型示意圖**：
```
┌─────────────────────── margin ───────────────────────┐
│  ┌──────────────────── border ─────────────────────┐ │
│  │  ┌───────────────── padding ──────────────────┐ │ │
│  │  │                                            │ │ │
│  │  │            content (內容)                  │ │ │
│  │  │                                            │ │ │
│  │  └────────────────────────────────────────────┘ │ │
│  └───────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────┘
```

---

#### 陰影效果

```css
.card {
    /* 盒子陰影 */
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
    /* 水平偏移 垂直偏移 模糊半徑 顏色 */

.text {
    /* 文字陰影 */

}
```

---

#### 轉換與動畫

```css
.button {
    /* 過渡效果 */
    transition: all 0.3s ease;
}

.button:hover {
    /* 滑鼠懸停時的變化 */
    transform: translateY(-3px);    /* 向上移動 */

.card {
    /* 更多變換效果 */
    transform: scale(1.1);          /* 放大 110% */

/* 動畫定義 */
@keyframes fadeIn {
    from {
        opacity: 0;
        transform: translateY(20px);
    }
    to {
        opacity: 1;
        transform: translateY(0);
    }
}

.animate {
    animation: fadeIn 0.5s ease-in;
}
```

## 📐 第四單元：響應式設計與排版

### 什麼是響應式設計（RWD）？

**定義**：網頁能夠自動適應不同螢幕大小的設計方式

**為什麼重要？**
- 手機使用者越來越多
- 提升使用者體驗
- Google SEO 會考量行動版體驗

**實現方式**：
1. 使用相對單位（%、rem、em）
2. 使用 Flexbox 和 Grid
3. 使用 Media Queries（媒體查詢）

---

### CSS 單位說明

#### 絕對單位
```css
.box {
    width: 300px;      /* 像素（固定大小）*/
    height: 200px;
}
```

#### 相對單位
```css
.box {
    width: 50%;        /* 父元素的 50% */
    font-size: 1.5rem; /* 相對於根元素的字體大小 */
    padding: 2em;      /* 相對於自己的字體大小 */
}

.container {
    max-width: 1200px; /* 最大寬度 */
    width: 100%;       /* 但不超過父元素 */
}
```

**建議用法**：
- 寬度：用 `%` 或 `max-width`
- 字體：用 `rem`
- 間距：用 `rem` 或 `em`

---

### Flexbox 彈性盒子

**基本概念**：
- 一維排版（橫向或縱向）
- 自動分配空間
- 輕鬆對齊元素

#### 容器屬性（父元素）

```css
.container {
    display: flex;              /* 啟用 Flexbox */
    
    /* 主軸方向 */
    flex-direction: row;        /* 預設：橫向 */
    flex-direction: column;     /* 縱向 */
    
    /* 主軸對齊 */
    justify-content: center;    /* 置中 */
    justify-content: space-between; /* 兩端對齊 */
    justify-content: space-around;  /* 平均分配 */
    
    /* 交叉軸對齊 */
    align-items: center;        /* 置中 */
    align-items: flex-start;    /* 靠上 */
    align-items: stretch;       /* 拉伸（預設）*/
    
    /* 換行 */
    flex-wrap: wrap;            /* 允許換行 */
    
    /* 間距 */
    gap: 1rem;                  /* 子元素之間的間距 */
}
```

#### 項目屬性（子元素）

```css
.item {
    flex: 1;              /* 平均分配空間 */
    flex-grow: 2;         /* 這個元素佔 2 份空間 */
    flex-shrink: 0;       /* 不縮小 */
}
```

#### 實戰範例：導覽列

```css
.navbar {
    display: flex;
    justify-content: space-between;  /* Logo 靠左，選單靠右 */
    align-items: center;             /* 垂直置中 */
}

.nav-menu {
    display: flex;
    gap: 2rem;                       /* 選單項目間距 */
}
```

---

### Grid 網格系統

**基本概念**：
- 二維排版（橫向和縱向）
- 適合複雜佈局
- 可以重疊元素

#### 容器屬性

```css
.grid-container {
    display: grid;
    
    /* 定義欄位 */
    grid-template-columns: 1fr 2fr 1fr;  /* 3 欄，中間較寬 */
    grid-template-columns: repeat(3, 1fr); /* 3 欄，等寬 */
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr)); /* 自動調整 */
    
    /* 定義列 */
    grid-template-rows: 100px auto 50px;
    
    /* 間距 */
    gap: 2rem;                /* 行列間距 */
    column-gap: 1rem;         /* 欄間距 */
    row-gap: 1rem;            /* 列間距 */
}
```

#### 項目屬性

```css
.grid-item {
    grid-column: 1 / 3;       /* 佔據第 1 到第 3 欄 */
    grid-row: 1 / 2;          /* 佔據第 1 列 */
}
```

#### 實戰範例：相簿網格

```css
.gallery-grid {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
    gap: 1.5rem;
}

.gallery-item {
    aspect-ratio: 4 / 3;      /* 保持比例 */
    overflow: hidden;
    border-radius: 12px;
}

.gallery-item img {
    width: 100%;
    height: 100%;
    object-fit: cover;        /* 裁切填滿 */
}
```

### Media Queries（媒體查詢）

**基本語法**：
```css
/* 桌機版樣式（預設）*/
.container {
    max-width: 1200px;
    display: flex;
}

/* 平板版（螢幕寬度小於 768px）*/
@media (max-width: 768px) {
    .container {
        flex-direction: column;
    }
    
    .hero h2 {
        font-size: 2rem;
    }
}

/* 手機版（螢幕寬度小於 480px）*/
@media (max-width: 480px) {
    .navbar {
        padding: 0.5rem;
    }
    
    .nav-menu {
        flex-direction: column;
        gap: 1rem;
    }
}
```

**常用斷點（Breakpoints）**：
- 手機：`max-width: 480px`
- 平板：`max-width: 768px`
- 桌機：`max-width: 1024px`
- 大螢幕：`min-width: 1200px`

### 響應式設計最佳實踐

#### 1. Mobile First（手機優先）

```css
/* 先寫手機版樣式 */
.container {
    padding: 1rem;
}

/* 再用 min-width 擴充到大螢幕 */
@media (min-width: 768px) {
    .container {
        padding: 2rem;
    }
}
```

#### 2. 使用相對單位

```css
/* ❌ 不好的做法 */
h1 {
    font-size: 32px;
}

/* ✅ 好的做法 */
h1 {
    font-size: 2rem;  /* 可隨根字體大小調整 */
}
```

#### 3. 圖片響應式

```css
img {
    max-width: 100%;   /* 不超過父元素寬度 */
    height: auto;      /* 保持比例 */
}
```

#### 4. 容器限制寬度

```css
.container {
    max-width: 1200px;  /* 不會太寬 */
    margin: 0 auto;     /* 置中 */
    padding: 0 20px;    /* 兩側留白 */
}
```

#### 3. JavaScript 優化

```javascript
// 減少 DOM 操作次數
// ❌ 不好（每次都查詢 DOM）
for (let i = 0; i < 100; i++) {
    document.getElementById('box').style.width = i + 'px';
}

// ✅ 好（先取得元素）
const box = document.getElementById('box');
for (let i = 0; i < 100; i++) {
    box.style.width = i + 'px';
}

// 使用事件委派
// ❌ 不好（每個項目都加監聽器）
document.querySelectorAll('.item').forEach(item => {
    item.addEventListener('click', handleClick);
});

// ✅ 好（只在父元素加一個監聽器）
document.querySelector('.list').addEventListener('click', function(e) {
    if (e.target.classList.contains('item')) {
        handleClick(e);
    }
});
```

### 常見學生問題與解答

#### Q1: 為什麼我的 CSS 沒有生效？

**可能原因與解決方法**：

1. **檔案連結錯誤**
   ```html
   <!-- ❌ 錯誤 -->
   <link rel="stylesheet" href="styles.css">
   
   <!-- ✅ 檢查檔名是否一致 -->
   <link rel="stylesheet" href="style.css">
   ```

2. **選擇器寫錯**
   ```css
   /* HTML 中是 class="card" */
   
   /* ❌ 錯誤（忘記加點）*/
   card {
       color: red;
   }
   
   /* ✅ 正確 */
   .card {
       color: red;
   }
   ```

3. **屬性拼字錯誤**
   ```css
   /* ❌ 錯誤 */
   .box {
       bakground-color: blue;  /* 拼錯字 */
   }
   
   /* ✅ 正確 */
   .box {
       background-color: blue;
   }
   ```

**除錯步驟**：
1. 用開發者工具檢查元素
2. 查看 CSS 是否被載入（Network 分頁）
3. 檢查是否有語法錯誤（Console 分頁）

---

#### Q2: 圖片為什麼不顯示？

**檢查清單**：

1. **檔案路徑**
   ```html
   <!-- 檔案結構：
   my-website/
   ├── index.html
   └── images/
       └── photo.jpg
   -->
   
   <!-- ✅ 正確 -->
   <img src="images/photo.jpg" alt="照片">
   
   <!-- ❌ 錯誤 -->
   <img src="photo.jpg" alt="照片">
   ```

2. **檔案名稱大小寫**
   ```html
   <!-- 檔案名稱是 Photo.jpg -->
   
   <!-- ❌ 錯誤（Linux/Mac 會區分大小寫）-->
   <img src="images/photo.jpg" alt="照片">
   
   <!-- ✅ 正確 -->
   <img src="images/Photo.jpg" alt="照片">
   ```

3. **使用開發者工具檢查**
   - 在圖片上按右鍵 → 檢查
   - 查看 Console 是否有 404 錯誤

---

#### Q3: JavaScript 沒有執行？

**可能原因**：

1. **忘記等待 DOM 載入**
   ```javascript
   // ❌ 錯誤（DOM 可能還沒載入完成）
   const button = document.getElementById('myButton');
   button.addEventListener('click', ...);
   
   // ✅ 正確
   document.addEventListener('DOMContentLoaded', function() {
       const button = document.getElementById('myButton');
       button.addEventListener('click', ...);
   });
   ```

2. **有語法錯誤**
   ```javascript
   // ❌ 錯誤（忘記關閉括號）
   function greet() {
       console.log('Hello';
   }
   
   // ✅ 正確
   function greet() {
       console.log('Hello');
   }
   ```

3. **ID 或 Class 寫錯**
   ```javascript
   // HTML 中是 id="myButton"
   
   // ❌ 錯誤
   const button = document.getElementById('myBtn');
   
   // ✅ 正確
   const button = document.getElementById('myButton');
   ```

**除錯方法**：
- 打開 Console 查看錯誤訊息
- 使用 `console.log()` 測試程式碼是否執行

---

#### Q4: 怎麼讓元素水平置中？

**不同情況的解決方法**：

1. **文字置中**
   ```css
   .text {
       text-align: center;
   }
   ```

2. **區塊元素置中**
   ```css
   .box {
       width: 500px;
       margin: 0 auto;  /* 上下 0，左右 auto */
   }
   ```

3. **Flexbox 置中**
   ```css
   .container {
       display: flex;
       justify-content: center;  /* 水平置中 */
       align-items: center;      /* 垂直置中 */
   }
   ```

4. **Grid 置中**
   ```css
   .container {
       display: grid;
       place-items: center;  /* 水平和垂直都置中 */
   }
   ```
