## 基礎修改

### 1. 修改個人資訊
```
請幫我把 index.html 中的「王小明」改成「我的名字」，
並把關於我的介紹改成：「我是一個喜歡畫畫和打籃球的高中生」
```

### 2. 更換配色
```
請幫我修改 style.css 中的主題顏色，
把紫藍色改成清新的綠色系配色
```

### 3. 新增興趣卡片
```
請在興趣區塊新增一張卡片，
圖示用 🎮，標題是「電玩遊戲」，
內容是「喜歡玩動作冒險類遊戲」
```

### 4. 修改照片數量
```
請把相簿區塊的照片從 6 張改成 9 張，
並調整成 3x3 的網格排列
```

## 新增功能

### 5. 加上返回頂部按鈕
```
請幫我加一個「返回頂部」按鈕，
當使用者往下捲動超過 500px 時出現，
點擊後平滑捲動回頁面最上方
```

### 6. 加上技能進度條
```
在關於我區塊新增一個「技能」小節，
加上 4 個技能的進度條動畫，
例如：程式設計 75%、英文 60%、繪畫 80%、音樂 50%
```

### 7. 加上計數器動畫
```
在首頁英雄區塊下方加上 3 個計數器，
分別顯示「學習時數」、「完成專案」、「咖啡杯數」，
數字要有從 0 開始跳動的動畫效果
```

### 8. 加上打字機效果
```
請在首頁的副標題加上打字機效果，
讓文字像是被一個字一個字打出來的樣子
```

### 9. 加上時間軸
```
新增一個「學習歷程」區塊，
用垂直時間軸的方式呈現我的學習經歷，
例如 2024 年開始學程式、2025 年學會做網頁等
```

### 10. 加上粒子背景
```
請在首頁的英雄區塊加上浮動粒子背景效果，
讓背景有一些小圓點在緩慢移動
```

## 美化特效

### 11. 加上滑鼠游標特效
```
請加上自訂滑鼠游標效果，
當滑鼠移動時有彩色軌跡跟隨
```

### 12. 卡片翻轉效果
```
請讓興趣卡片變成可以翻轉的，
正面顯示圖示和標題，背面顯示詳細內容
```

### 13. 視差捲動效果
```
請在首頁英雄區塊加上視差捲動效果，
讓背景移動速度比前景慢，產生立體感
```

### 14. 文字漸層色彩
```
請把首頁標題改成彩色漸層文字，
從藍色漸層到紫色，並加上閃爍動畫
```

### 15. 照片濾鏡效果
```
請在相簿的照片上加上黑白濾鏡，
當滑鼠懸停時變回彩色，並加上平滑過渡
```

### 16. 載入動畫
```
請加上網頁載入動畫，
在所有內容載入前顯示一個旋轉的 Logo
```

### 17. 加上雪花特效
```
請在整個網頁加上飄落的雪花動畫效果，
雪花要有不同大小和速度
```

### 18. 按鈕波紋效果
```
請在所有按鈕上加上點擊波紋效果，
就像水波紋從點擊位置擴散出去
```

## 疑難排解

### 19. 修正手機版顯示問題
```
我的網頁在手機上看起來太擠了，
請幫我優化手機版的排版，
讓文字和按鈕都變得更大更好點
```

### 20. 加速載入速度
```
我的網頁載入速度有點慢，
請幫我優化 CSS 和 JavaScript，
移除不必要的程式碼
```

### 21. 修正導覽列問題
```
當我捲動頁面時，導覽列會擋住內容，
請調整錨點連結，讓捲動時能正確顯示區塊標題
```

### 22. 深色模式優化
```
深色模式下有些文字看不清楚，
請調整深色模式的顏色對比，
確保所有文字都清晰可讀
```

## 💡 使用 AI 的小技巧

### ✅ 好的提示詞範例：
- **具體明確**：「請把首頁標題改成藍色，字體大小 3rem」
- **提供脈絡**：「請在關於我區塊下方新增一個技能區塊」
- **說明需求**：「請加上淡入動畫，讓卡片從左到右依序出現」

### ❌ 不好的提示詞範例：
- **太模糊**：「讓網頁變好看」
- **太複雜**：一次要求修改 10 個以上的東西
- **沒有位置**：「加一個按鈕」（沒說要加在哪裡）

### 🎯 進階技巧

1. **分步驟進行**：不要一次要求太多功能，一次做一個
2. **測試結果**：每次修改後都要預覽，確認效果
3. **保留備份**：重大修改前，先複製一份檔案
4. **學習理解**：不要只是複製貼上，試著理解 AI 產生的程式碼
5. **持續優化**：發現問題就立刻請 AI 協助修正

## 🚀 挑戰題

當你熟悉基本功能後，可以嘗試這些進階挑戰：

1. **響應式導覽選單**：在手機版時變成漢堡選單
2. **多語言切換**：加上中英文切換功能
3. **聯絡表單驗證**：加上完整的表單驗證（email 格式、必填欄位等）
4. **相簿燈箱效果**：點擊照片後用燈箱模式顯示大圖
5. **音樂播放器**：嵌入一個簡單的音樂播放器
6. **遊戲彩蛋**：加上 Konami Code（上上下下左右左右BA）秘技
