## 🚀 快速開始

### 在 GitHub Codespaces 中使用

1. **開啟專案**
   - 點擊綠色的「Code」按鈕
   - 選擇「Codespaces」分頁
   - 點擊「Create codespace on main」
   - 等待環境建立完成（約 1-2 分鐘）

2. **啟動預覽**
   - 在左側找到 `index.html`
   - 右鍵點擊選擇「Open with Live Server」
   - 網頁會在右側視窗開啟

3. **開始學習**
   - 閱讀 `TUTORIAL.md` 了解基礎知識
   - 完成 `MISSIONS.md` 中的任務挑戰
   - 參考 `AI-PROMPTS.md` 學習如何使用 AI

## 🎨 客製化你的網頁

### 修改配色

開啟 `style.css`，在最上方找到 CSS 變數：

```css
:root {
    --primary-color: #6366f1;  /* 改成你喜歡的顏色 */
    --text-primary: #1f2937;   /* 文字顏色 */
    --bg-primary: #ffffff;     /* 背景顏色 */
}
```

### 修改內容

開啟 `index.html`，直接修改：
- 個人資訊（名字、介紹）
- 興趣內容
- 相簿照片
- 聯絡方式

### 新增功能

使用 AI 協助你新增功能,例如：
```
請在網頁加上一個「返回頂部」按鈕,
當使用者往下捲動超過 300px 時出現,
點擊後平滑捲動回頂部
```

## 💡 使用 AI 的技巧

### ✅ 好的提示詞
```
請在關於我區塊下方新增一個技能展示區塊,
包含 5 個技能的進度條,每個進度條要有從 0 開始的填充動畫
```

### ❌ 不好的提示詞
```
加東西
```

### 關鍵要點
1. **說明位置**：在哪個區塊、哪個檔案
2. **描述需求**：要什麼功能、什麼效果
3. **提供細節**：顏色、大小、動畫等

更多範例請參考 [`AI-PROMPTS.md`](AI-PROMPTS.md)

## ❓ 常見問題

### Q: 我完全沒有程式基礎,可以學會嗎?
**A**: 當然可以!這個專案就是為零基礎設計的。跟著 `TUTORIAL.md` 一步步學習,遇到問題就問 AI,很快就能上手！

### Q: 我需要安裝什麼軟體嗎?
**A**: 不需要!在 GitHub Codespaces 中所有工具都已經預先安裝好了,打開就能用。

### Q: 我改壞了怎麼辦?
**A**: 
1. 使用 `Ctrl+Z` 復原
2. 在 Source Control 中查看變更並復原
3. 最壞情況下可以重新 Fork 一份專案

### Q: 如何分享我的網頁給朋友看?
**A**: 
1. 在 Codespaces 中,網頁會自動產生一個預覽網址
2. 可以暫時分享這個網址給朋友
3. 之後學習如何部署到 GitHub Pages 就能有永久網址

### Q: 完成專案後可以做什麼?
